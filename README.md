# Tecnologias utilizadas
- HTML / CSS / JS / TypeScript
- React
- Tailwind CSS
- Shadcn

## Implantação em produção
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/RafaelPeral/Portifolio

# instalar dependências
npm install

# executar o projeto
npm run dev
```
